# Explorador DOM y Dashboard de Usuarios

Este proyecto es una práctica de desarrollo web utilizando **HTML, CSS y JavaScript** para demostrar la manipulación del **DOM**, el uso de **eventos**, consumo de **API con Fetch**, y la implementación de **componentes interactivos** en una página web.

El proyecto incluye una lista de tareas dinámica, un dashboard que obtiene usuarios desde una API pública y varios componentes de interfaz construidos con JavaScript.

---

## Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript (ES6)
* Fetch API
* MutationObserver
* JSONPlaceholder API

---

## Funcionalidades del proyecto

### Lista de tareas

* Permite agregar tareas dinámicamente.
* Las tareas pueden marcarse como **completadas** al hacer clic.
* Se pueden eliminar con el botón de eliminar.
* Contador automático de tareas.
* Cambio de color del título con doble clic.

### Dashboard de usuarios

* Obtiene usuarios desde una API externa.
* Muestra información básica de cada usuario.
* Permite ver publicaciones relacionadas con cada usuario.
* Sistema de búsqueda para filtrar usuarios en tiempo real.

### Componentes interactivos

El proyecto también incluye algunos componentes de interfaz creados con JavaScript:

* **Tabs**: permite cambiar entre diferentes secciones de contenido.
* **Acordeón**: muestra y oculta información en bloques desplegables.
* **Modal**: ventana emergente con contenido dinámico.
* **Carrusel**: cambio automático de slides.

---

## API utilizada

Los datos del dashboard se obtienen desde la API pública:

https://jsonplaceholder.typicode.com

Se utilizan los endpoints:

* `/users`
* `/posts`
